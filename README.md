# IntelliSOC

Intelligent Security Operations Center

## Overview

This project implements an intelligent SOC system for log analysis, threat detection, and incident response.

## Structure

- core/: Core modules for ingestion, normalization, detection, incidents, intelligence, and database.
- data/: Data directories for raw and normalized logs, datasets.
- rules/: JSON files for detection rules.
- web/: Web interface with templates and static files.
- reports/: Generated reports.
